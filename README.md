1) What is the difference between null and undefined?
undefined মানে value দেওয়া হয়নি,
null → মানে ইচ্ছা করে খালি রাখা হয়েছে।

2) What is the use of the map() function in JavaScript? How is it different from forEach()?
map() → নতুন array return করে।
forEach() → নতুন array return করে না, শুধু loop চালায়।

3)  What is the difference between == and ===?
== (Loose equality) → শুধু value চেক করে, type চেক করে না।
=== (Strict equality) → value এবং type দুটোই চেক করে।

4) What is the significance of async/await in fetching API data?
async/await asynchronous কোডকে সহজ ও readable করে।
Promise handle করা সহজ হয়।
Error handle করতে try...catch ব্যবহার করা যায়।

5) Explain the concept of Scope in JavaScript (Global, Function, Block).
Scope মানে ভ্যারিয়েবল কোথায় access করা যাবে।
Global Scope → পুরো প্রোগ্রামে ব্যবহার করা যায়।
Function Scope → শুধু ওই function এর ভিতরে ব্যবহার করা যায়।
Block Scope → { } এর ভিতরে (let, const দিয়ে)।
